# Tic-Tac-Toe Game (C++)

A text-based Tic-Tac-Toe game developed in C++ using Visual Studio Code. This project runs entirely within the console/terminal.

## Features
* Classic 3x3 local grid.
* Simple coordinate input for players.
* Automatic win and draw validation.

## How to Run
1. Download or clone this repository.
2. Open `tictactoe.cpp` in Visual Studio Code.
3. Ensure you have a C++ compiler installed (like MinGW/GCC).
4. Run the code using the terminal or the C++ Runner extension.
